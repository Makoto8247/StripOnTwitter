# StripOnTwitter
七夕の日にTwitterで短冊をツイートするためのものです。<br>
願いは20文字まで入力してください。<br>
名前は10文字まで入力してください。<br>
小文字アルファベットは大文字になります。<br>
表示結果(例):<br>
┏━━━━━┓<br>
┃　　〇　　┃<br>
┃　　　　　┃<br>
┃　　願　　┃<br>
┃　　い　　┃<br>
┃　　を　　┃<br>
┃　　表　　┃<br>
┃　　示　　┃<br>
┃名　　　　┃<br>
┃前　　　　┃<br>
┃を　　　　┃<br>
┃表　　　　┃<br>
